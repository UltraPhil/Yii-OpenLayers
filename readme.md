Yii-OpenLayers
=====

What is Yii-OpenLayers?
--------------
Yii-OpenLayers is an extension to include OpenLayers map, OOP-style, PHP, in your YII applications.

What I'm planning to support:
- Map
	- Layers
	- LonLat items
	- Markers
- OL Widget
- JS scripts to help interracting with the map.
